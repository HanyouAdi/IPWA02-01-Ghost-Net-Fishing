Melde- und Koordinierungsplattform
zur Bergung von Geisternetzen

Was möchten Sie tun?

Wählen Sie eine Aktion aus. Die Plattform führt Sie anschließend durch den passenden Ablauf.

Hinweis: Die Auswahl bestimmt, welche Geisternetze angezeigt werden und welche Statusänderung anschließend gespeichert wird.

So funktioniert die Plattform

Geisternetz melden

Nutzen Sie diese Aktion, wenn Sie ein neues Geisternetz entdeckt haben. Erfasst werden Position, geschätzte Größe und optional Kontaktdaten.

Status: Gemeldet

Bergung übernehmen

Nutzen Sie diese Aktion, wenn Sie die Bergung eines oder mehrerer gemeldeter Geisternetze übernehmen möchten.

Status: Bergung bevorstehend

Als geborgen melden

Nutzen Sie diese Aktion nach erfolgreicher Bergung. Angezeigt werden nur Geisternetze, für deren Bergung Sie eingetragen wurden.

Status: Geborgen

Als verschollen melden

Nutzen Sie diese Aktion, wenn ein gemeldetes oder zur Bergung vorgesehenes Geisternetz nicht mehr auffindbar ist.

Status: Verschollen
